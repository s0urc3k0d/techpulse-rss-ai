## Specific ARIA Implementation

### Interactive Elements

```tsx
// Buttons with loading states
<button 
  aria-busy={loading}
  aria-label="Lancer le scraping"
  disabled={loading}
>
  {loading ? 'Scraping en cours...' : 'Lancer'}
</button>

// Checkboxes
<input 
  type="checkbox"
  aria-label={`Sélectionner ${article.title}`}
/>

// Form inputs
<input 
  id="start-date"
  aria-label="Sélectionner la date de début"
/>
<label htmlFor="start-date">Date de début</label>

// Links
<a 
  href={url}
  aria-label={`Lire l'article: ${title}`}
  target="_blank"
  rel="noopener noreferrer"
>
  Lire
</a>
```
